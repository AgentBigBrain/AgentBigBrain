# Support

## Getting Help

### 💬 Questions & Usage Help

- **Preferred:** [GitHub Discussions](https://github.com/AgentBigBrain/AgentBigBrain/discussions) (if enabled)
- **Alternative:** Open a [GitHub issue](https://github.com/AgentBigBrain/AgentBigBrain/issues/new) with the `[Question]` prefix

When asking questions, include:

- What you're trying to accomplish
- Commands you ran and their output
- Expected vs. actual behavior
- Your environment (OS, Node.js version)
- Whether the issue reproduces with `BRAIN_MODEL_BACKEND=mock`

### 🐛 Bug Reports

Open a [GitHub issue](https://github.com/AgentBigBrain/AgentBigBrain/issues/new) with the `[Bug]` prefix. See [CONTRIBUTING.md](CONTRIBUTING.md#-bug-reports) for what to include.

### 💡 Feature Requests

Open a [GitHub issue](https://github.com/AgentBigBrain/AgentBigBrain/issues/new) with the `[Feature]` prefix. See [CONTRIBUTING.md](CONTRIBUTING.md#-feature-requests) for what to include.

### 🔒 Security Issues

**Do not report vulnerabilities publicly.** Use the private reporting channels in [SECURITY.md](SECURITY.md).

## Response Expectations

| Channel | Target Response |
|---|---|
| Security reports | Within 72 hours |
| Bug reports | Within 7 calendar days |
| Feature requests | Best effort |
| Questions | Best effort |

All support is provided on a best-effort basis by maintainers and community volunteers.
